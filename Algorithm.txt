Приложение должно подготавливать данные датчиков кислорода, температуры, давления, времени работы насосов из файлов с
опытного участка в формате csv: заполнение пробелов в данных (установкой средних значений), первичный анализ информации.
Файлы подготовлены посуточно.
Предварительная подготовка файла с данными:
1. Удаление повторяющихся строк (названия столбцов).
2. Удаление строк с пропущенными значениями в случае нестандартного поведения SCADA. Удаление строк с NaN.
3. Конвертация значений столбцов во float для последующих расчетов.
4. Заполнения строк с нулевым значением для столбцов O2, T7, T6, T5, T4, T3, T2, T1.
